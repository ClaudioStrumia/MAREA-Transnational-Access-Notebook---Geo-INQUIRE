# MAREA-Transnational-Access-Notebook---Geo-INQUIRE
Notebook based on the outcomes of the MAREA Transnational Access project funded by GeoIqnuire. Geo-INQUIRE is funded by the European Commission under project number 101058518 within the HORIZON-INFRA-2021-SERV-01 call.

The /codes folder contains two Jupyter notebooks.
With the first it is possible to visualize a seismic catalogue of events occurred in the area close to the fiber optic cables forming the ABYSS network in Chile, select one event, and use the magnitude laws calibrated within the MAREA project to evaluate the magnitude. The DAS data are publicly available and downloaded from the FDSN DAS data stream under the network code F1 (Rivet et al., 2025 https://doi.org/10.15778/RESIF.F1).
The second notebook is sued to visualize the outcomes of the magnitude computation.

The .grd files necessary for the "Plot_F1_mag.ipynb" notebook can be downloaded from Zenodo repo: https://doi.org/10.5281/zenodo.21360273
